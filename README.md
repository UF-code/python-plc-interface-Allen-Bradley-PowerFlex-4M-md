# PLC Interface Allen-Bradley PowerFlex 4M
Monitoring & Controlling the Device Resources

## Tech Stack
> Python, PyQT5, PyQtGraph, PyModbus
### Project Setup & How To Compile n Run
```
pip install pyqt5
pip install pymodbus
pip install pyqtgraph
python <filename>.py
```
